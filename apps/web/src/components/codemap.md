# apps/web/src/components/

## Responsibility
React components for the web frontend, including shadcn/ui primitives and domain-specific components.

## Files
- `pocketbase-provider.tsx` — React context provider for PocketBase client instance
- `providers.tsx` — Aggregated providers wrapper (theme, auth, PocketBase)
- `theme-toggle.tsx` — Dark/light mode toggle
- `verse-badge.tsx` — Badge displaying a Bible verse reference
- `verse-content.tsx` — Verse text display component
- `verse-input.tsx` — Input for entering Bible verse references
- `campus-input.tsx` — Autocomplete input for campus selection
- `pastor-input.tsx` — Autocomplete input for pastor selection
- `ui/` — shadcn/ui primitive components (button, card, dialog, etc.)
