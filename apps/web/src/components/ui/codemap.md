# apps/web/src/components/ui/

## Responsibility
shadcn/ui primitive components — unstyled, composable UI primitives built on Radix UI.

## Files
- `badge.tsx` — Inline badge/tag component
- `button.tsx` — Button with variants (default, destructive, outline, secondary, ghost, link)
- `card.tsx` — Card container with Header, Title, Description, Content, Footer subcomponents
- `dialog.tsx` — Modal dialog with Trigger, Content, Header, Footer, Title, Description
- `input.tsx` — Text input with error styling
- `label.tsx` — Form label
- `select.tsx` — Select dropdown with Trigger, Value, Content, Item, Group
- `sheet.tsx` — Slide-out panel with Trigger, Content, Header, Footer, Title, Description, Side variants
- `tabs.tsx` — Tab navigation with List, Trigger, Content
- `textarea.tsx` — Multi-line text input
- `toggle-group.tsx` — Group of toggle buttons (single/multi select)
- `toggle.tsx` — Single toggle button with pressed state
